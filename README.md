# 🌟 Números 1-9

App educativa para aprender los números del 1 al 9, diseñada para Claudia.  
Compatible con iPad y Apple Pencil.

## Actividades

- 🎹 **Piano** — teclas del 1 al 9 con escala musical Do→Mi
- 🍎 **Cuenta** — cuenta figuras y elige el número correcto
- ➡️ **¿Cuál sigue?** — adivina el número siguiente
- 🐘 **¿Cuál es mayor?** — elige el número más grande
- 🐭 **¿Cuál es menor?** — elige el número más pequeño
- 🔢 **Ordena** — toca los números en orden
- ➕ **Sumas** — sumas sencillas hasta el 9
- ➖ **Restas** — restas sin resultado negativo
- 🔀 **Mezcla** — sumas y restas combinadas
- ✏️ **Repasa** — traza los números con Apple Pencil
